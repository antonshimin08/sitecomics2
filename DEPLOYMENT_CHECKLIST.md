## 📝 PRODUCTION DEPLOYMENT STEPS

### 1. Подготовка сервера

```bash
# Обновить PHP до 8.1+
php -v

# Включить необходимые расширения
# PDO, SQLite (или MySQL)

# Установить зависимости
composer install --no-dev

# Установить права на папки
chmod 755 /var/www/comic-universe
chmod 777 /var/www/comic-universe/logs
chmod 777 /var/www/comic-universe/cache
chmod 777 /var/www/comic-universe/uploads
```

### 2. Миграция БД

```bash
# Экспортировать данные из SQLite
sqlite3 database.sqlite ".dump" > backup.sql

# Импортировать в MySQL
mysql -u root -p comic_universe < backup.sql

# Или использовать скрипт миграции
php migrate.php sqlite:mysql
```

### 3. Конфигурация

```bash
# Скопировать config.php.example в config.php
cp config.php.example config.php

# Отредактировать config.php
# - ENVIRONMENT = 'production'
# - DEBUG_MODE = false
# - DB_TYPE = 'mysql'
# - DB_HOST, DB_NAME, DB_USER, DB_PASS
# - SITE_URL = 'https://yourdomain.com/'

nano config.php
```

### 4. SSL/HTTPS

```bash
# Установить Let's Encrypt сертификат
sudo certbot certonly --webroot -w /var/www/comic-universe -d yourdomain.com

# Обновить .htaccess для редиректа HTTP → HTTPS
# Раскомментировать строки редиректа
```

### 5. Развёртывание

```bash
# Загрузить файлы на сервер
git clone https://github.com/antonshimin08/sitecomics2.git /var/www/comic-universe

# или

scp -r ./* user@server:/var/www/comic-universe/

# Установить права
chmod 755 /var/www/comic-universe -R
chmod 777 /var/www/comic-universe/logs
chmod 777 /var/www/comic-universe/cache
chmod 777 /var/www/comic-universe/uploads
```

### 6. Финальная проверка

```bash
# Проверить, что сайт работает
curl https://yourdomain.com/

# Проверить API
curl https://yourdomain.com/api/read.php

# Проверить логи на ошибки
tail -f /var/www/comic-universe/logs/*.log
```

## 🚨 TROUBLESHOOTING

### "Database connection failed"

- [ ] Проверить конфигурацию БД в `config.php`
- [ ] Убедиться, что БД существует и доступна
- [ ] Проверить права доступа на файл БД (SQLite)

### "Fatal error: Class not found"

- [ ] Запустить `composer install`
- [ ] Проверить, что пути к файлам правильные
- [ ] Проверить `autoload.php`

### "Permission denied"

- [ ] Изменить права на папки: `chmod 777 logs cache uploads`
- [ ] Проверить владельца файлов: `chown -R www-data:www-data /var/www/comic-universe`

### "500 Internal Server Error"

- [ ] Включить DEBUG_MODE для просмотра ошибки
- [ ] Проверить логи сервера
- [ ] Проверить логи приложения в `/logs`

### ".htaccess not working"

- [ ] Убедиться, что `mod_rewrite` включен: `a2enmod rewrite`
- [ ] Перезагрузить Apache: `systemctl restart apache2`
- [ ] Проверить, что `.htaccess` находится в корне проекта
