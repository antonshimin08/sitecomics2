🚀 БЫСТРЫЙ СТАРТ

  DEVELOPMENT:
  ─────────────────────────────────────────────────────────────
  1. composer install
  2. php setup-db.php
  3. php -S localhost:8000
  ─────────────────────────────────────────────────────────────

  PRODUCTION (Linux/Mac):
  ─────────────────────────────────────────────────────────────
  1. bash deploy.sh
  2. nano .env (отредактировать конфиг)
  3. curl https://domain.com/deploy-status.php
  ─────────────────────────────────────────────────────────────

  PRODUCTION (Windows):
  ─────────────────────────────────────────────────────────────
  1. powershell -ExecutionPolicy Bypass -File deploy.ps1
  2. Отредактировать .env
  3. Открыть https://domain.com/deploy-status.php в браузере
  ─────────────────────────────────────────────────────────────

🎯 СЛЕДУЮЩИЕ ШАГИ

  1. 📝 Отредактировать .env
     - DB_HOST, DB_USER, DB_PASS
     - SITE_URL (домен)

  2. 🌐 Настроить веб-сервер

✨ ДОПОЛНИТЕЛЬНО

  Для автоматического развёртывания используйте:
  
  Linux/Mac:
    bash deploy.sh

  Windows (PowerShell):
    powershell -ExecutionPolicy Bypass -File deploy.ps1

  Эти скрипты автоматически:
  - Проверяют PHP версию и расширения
  - Создают .env файл
  - Создают необходимые папки
  - Устанавливают зависимости (composer)
  - Инициализируют БД
  - Запускают проверку статуса
